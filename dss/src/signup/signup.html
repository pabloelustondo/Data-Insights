Mobicontrol Instance Enrollment

As a MobiControl administrator, you can use this page to enroll an internet-exposed MobiControl instance into our Data Analytics service. This enrollment serves two pourposes: 1) Allow valid mobicontrol user from that MobiControl instance to access our Data Analytics Dashboard and APIs. 2) Allows mobicontrol administrators to setup mobicontrol data extraction process that will populate your data analytics store.
